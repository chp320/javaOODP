- 데코레이터 패턴은 컴포지트 패턴과 비슷한 다이어그램을 가진다. (https://keichee.tistory.com/180)

2. 목적
- Attach additional responsibilities to an object dynamically. Decorators provide a flexible alternative to subclasssing for extending functionality.
- 객체에 동적으로 추가적인 기능을 추가한다. Decorator는 확장 기능에 대해 subclass 추가하는 방법으로 유연함을 제공한다.
  ( Decorator는 데코레이터 패턴의 '메인'으로 ConcreteDecorator를 관리한다. )

3. 예제
- 크리스마스트리를 기본으로 두고 장식을 위한 여러 ConcreteDecorator를 추가한다.
